## 💻 Quick Start

### Local Development
1. Clone the repository:
```bash
git clone https://github.com/wajeehudin663-a11y/portfolio.git
cd portfolio
```

2. Open `index.html` in your browser (or use a local server):
```bash
python -m http.server 8000
```

3. Visit `http://localhost:8000` in your browser

### Deployment
This portfolio is automatically deployed via GitHub Pages:
1. Push changes to the `main` branch
2. GitHub automatically builds and deploys
3. Your site appears at `yourusername.github.io/portfolio`

## 🎨 Customization

### Colors
Edit the CSS variables in `styles.css`:
```css
:root {
    --primary-color: #0066cc;
    --secondary-color: #00d4ff;
    --dark-bg: #0a0e27;
    /* ... more colors ... */
}
```

### Content
Edit the sections directly in `index.html`:
- Update personal information in the Hero section
- Modify projects in the Projects section
- Add your GitHub/LinkedIn links in the Contact section

### Styling
Modify `styles.css` to change:
- Colors and gradients
- Fonts and typography
- Spacing and layouts
- Responsive breakpoints
